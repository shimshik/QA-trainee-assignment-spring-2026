# Баг-репорт 1

**Описание ошибки:** Не корректно работает фильтр по ценовому диапозону

**Проект:** Модерация Авито

**Серьезность:** S2 Critical

**Приоритет:** High

**Статус:** New

**Автор:** Шимшилашвили М.И.

**Окружение:** Браузер Google Chrome v.142.0.7444.163(официальная сборка) 

**Шаги воспроизведения:**

1. Открыть страницу по адресу https://cerulean-praline-8e5aa6.netlify.app/
   
2. Ввести значения 1000 - 10000 в фильтр по ценовому диапозону
 
**Ожидаемый результат:** В выдаче будут только карточки объявлений в ценовом диапозоне 1000 - 10000

**Фактический результат:** В выдаче находятся карточки объявлений вне ценового диапозона, некоторые объявления,
входящие в ценовой диапозон не отображаются

**Дополнительно** В URI наблюдается неправильная передача GET-парамметров из фильтра ценового диапозона ,а именно
к минимальной и максимальной цене прибавляется 5000

# Баг-репорт 3

**Описание ошибки:** Не корректно работает включение автообновления статистики

**Проект:** Модерация Авито

**Серьезность:** S2 Critical

**Приоритет:** High

**Статус:** New

**Автор:** Шимшилашвили М.И.

**Окружение:** Браузер Google Chrome v.142.0.7444.163(официальная сборка) 

**Шаги воспроизведения:**

1. Открыть страницу по адресу https://cerulean-praline-8e5aa6.netlify.app/stats
   
2. Нажать кнопку остановки автоматического обновления статистики

3. Нажать кнопку запуска автоматического обновления статистики
**Ожидаемый результат:** Появится таймер автообноления с начальной точки отсчета - 5:00, 
сообщение "Автообновление отключено" пропадет

**Фактический результат:** Сообщение "Автообновление отключено" не пропадает, таймер автообновления не появляется
